# 球探比分——体育桌球场馆预约神器

  "球探比分——体育桌球场馆预约神器"应用是一款线上预约桌球场馆的app。
    
    该应用通过线上选择桌球馆，选择预定日期、选择场馆、添加预约人数、预约时长等，实现线上预定桌球馆的应用。
    同时在桌球场馆详情页面查看关于桌球馆的相关场馆设施、场馆服务、预定须知等。
    
    "首页"实现对桌球馆的分类，新店开业、top好店、热门场馆、天天特惠、今日优选和今日推荐等类型分类。
    
    意在给喜欢桌球项目的人，提供一个线上预约桌球馆的平台。
